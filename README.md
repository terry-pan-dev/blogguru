### Tech stack
- react
- react-redux
- react-router-dom
- immutablejs
- react-sagas (async call)
- styled-components (UI library)
- react-transition-group (animation)
- miragejs (data mock)

### features
- infinite scroll
- data mock (fake server)
- sign up
- blog detail
- writing blog
- go back to top
- login page
