## Blogguru

Blogguru is a blog writing platform purely built by using Reactjs.

### Tech stack
- react
- react-redux
- react-router-dom
- react-sagas (async call)
- styled-components (UI library)
- react-transition-group (animation)
- miragejs (data mock)

### features
- infinite scroll
- data mock (fake server)
- sign up
- blog detail
- writing blog
